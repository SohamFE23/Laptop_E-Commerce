# Laptop E-Commerce

A responsive, static front-end for an online laptop store built with HTML and CSS.

This repository contains the UI for a laptop e-commerce site. It focuses on layout, styling, and responsive design using plain HTML and CSS. The project is suitable as a portfolio piece or a starter template for static e-commerce pages.

## Features

- Product listing/grid layout
- Product details / cards (HTML structure)
- Responsive styles for desktop and mobile (CSS)
- Clean, semantic HTML and organized CSS

## Tech / Languages

- HTML
- CSS

## Getting started

Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

To view the project locally

1. Clone the repository:

   git clone https://github.com/SohamFE23/Laptop_E-Commerce.git
2. Open the project folder and open `index.html` in your browser.

Alternatively, you can serve the project with a simple static server, for example using Python:

- Python 3.x:

  python -m http.server 8000

Then open http://localhost:8000 in your browser.

## Project structure (example)

- index.html - main landing / product listing page
- css/ - stylesheet files (e.g. styles.css)
- assets/ - images and icons

Adjust paths if your repository uses a different structure.
